# GEM Landing Page - Proyecto SKILLS

Este proyecto contiene la landing page premium para la agencia **GEM**, enfocada en **META Ads** y **SEO**.

## Estructura del Proyecto
- `index.html`: Código fuente completo de la landing page (HTML5 + CSS Embebido).
- `stitch-prompt.txt`: El prompt detallado utilizado para generar este diseño en Google Stitch.

## Características Principales
- **Diseño Premium**: Modo oscuro con acentos turquesa/azul vibrante.
- **Copy Estratégico**: Basado en las mejores prácticas de Copywriting (Beneficios > Características).
- **Enfoque SEO**: Estructura de encabezados optimizada.
- **Enfoque META Ads**: Sección dedicada a escalabilidad y ROI.

## Instrucciones
Puedes abrir el archivo `index.html` directamente en tu navegador para visualizar el diseño final.
